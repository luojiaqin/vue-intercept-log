# vue-intercept-log

## 使用方法
```
npm install vue-intercept-log
```

## 功能（记录正式环境中不方便打印及调试时的调试日志）

* 拦截console正常异常打印
* 拦截http正常异常请求
* 监听按钮点击触发（未完成）
* 增加vue-log查看实时日志页面
* 存储在indexdb中（未完成）
* 下载日志（未完成）
* 读取当前环境配置
